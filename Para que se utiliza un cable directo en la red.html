Para que se utiliza un cable directo en la red


Un cable directo en redes se utiliza para conectar dispositivos diferentes entre sí. Aquí te explico sus principales usos y características: Definición: Un cable directo es aquel en el que los pines de ambos extremos del cable están conectados en el mismo orden. Usos principales: Conectar un ordenador a un switch o hub Conectar un router a un switch Conectar un servidor a un switch Configuración: Ambos extremos del cable siguen el mismo estándar de cableado, ya sea T568A o T568B. Dispositivos típicos: Conecta dispositivos de diferentes capas del modelo OSI, por ejemplo: Capa 1 (hub) con Capa 2 (switch) Capa 2 (switch) con Capa 3 (router) Longitud máxima: Generalmente limitada a 100 metros para cables de cobre Ethernet. Ventajas: Simple de fabricar Versátil para la mayoría de las conexiones en una red local Consideraciones: Es importante no confundir un cable directo con un cable cruzado, que se usa para conectar dispositivos similares.
